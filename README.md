# **CPU Scheduling and Memory Allocation Algorithms in Java**

## **Περιγραφή**
Πρόγραμμα σε **Java** που υλοποιεί ένα απλοποιημένο μοντέλο λειτουργικού συστήματος για τη μελέτη αλγορίθμων δρομολόγησης διεργασιών και δυναμικής τμηματοποίησης μνήμης. Προσομοιεί τη λειτουργία συστήματος με διεργασίες που μεταβαίνουν μεταξύ καταστάσεων (NEW, READY, RUNNING, TERMINATED) και ελέγχει τη συμπεριφορά αλγορίθμων υπό συνθήκες περιορισμένων πόρων.

---

### 🚀 **Project Overview**
- **Δρομολόγηση CPU**: Υλοποίηση αλγορίθμων **FCFS**, **SRTF** (με προεκχώρηση), και **Round Robin** (με προσαρμοσμένο κβάντο).
- **Διαχείριση Μνήμης**: Εφαρμογή στρατηγικών **Best Fit**, **Worst Fit**, **First Fit**, και **Next Fit** για την τοποθέτηση διεργασιών σε τμήματα RAM.
- **Καταστάσεις Διεργασιών**: Διαχείριση μεταβάσεων με χρόνους 1-2 κύκλων ρολογιού και προσομοίωση φαινομένων (στέρηση, φάλαγγα).

---

### 🔍 **Key Features**
- **Δυναμική Διαχείριση Πόρων**: Τοποθέτηση διεργασιών σε διαθέσιμα τμήματα μνήμης με τη χρήση της κλάσης `MMU`.
- **Χρονικοί Κύκλοι**: Προσομοίωση χρόνων μετάβασης (π.χ., 2 κύκλοι για εναλλαγή RUNNING / READY).
- **Κλάσεις Λειτουργικού Συστήματος**: `Process` (PID, καταιγισμοί CPU), `CPU` (διαχείριση εκτέλεσης), `MemorySlot` (τμήματα μνήμης).

---

### 🛠️ **Technical Highlights**
- **Προσομοίωση Συνθηκών**: Δοκιμές σε περιπτώσεις έλλειψης μνήμης, ταυτόχρονης άφιξης διεργασιών, και "κενών" εκτέλεσης.
- **Συμπεριφορά Συστήματος**: Ανίχνευση φαινομένων όπως παρατεταμένη στέρηση ή δημιουργία κενών μνήμης.
- **Επεκτασιμότητα**: Υποστήριξη πολλαπλών αλγορίθμων δρομολόγησης/τοποθέτησης με βάση αφηρημένες κλάσεις (`Scheduler`, `MemoryAllocationAlgorithm`).

---

### 📂 **Code Structure**
- **Process.java**: Διαχείριση διεργασιών (PID, καταιγισμοί CPU, απαιτήσεις μνήμης).
- **CPU.java**: Προσομοίωση εκτέλεσης διεργασιών και εναλλαγών με βάση αλγόριθμους δρομολόγησης.
- **MMU.java**: Διαχείριση τμημάτων μνήμης και εφαρμογή στρατηγικών τοποθέτησης.
